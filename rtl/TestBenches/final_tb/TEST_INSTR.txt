// ============================================================================
// Corrected instruction memory image for full ALU + data memory test
// ----------------------------------------------------------------------------
// - PC.v fetches 16-bit instructions as:
//       instr_o[7:0]  = instruction_mem[PC]
//       instr_o[15:8] = instruction_mem[PC+1]
//   so each instruction is stored as:
//       low byte first
//       high byte second
//
// - Instruction addresses are byte addresses.
//   Since each instruction is 2 bytes, PCs go 0, 2, 4, 6, ...
//
// - This program tests:
//       ADD, SUB, SLL, AND, ADDI, SW, LW, JMP
//
// - Final instruction is JMP -1 so the processor loops forever at PC = 30.
// ============================================================================

// PC = 0
// ADD r3, r1, r0

03
